# 因子注册及计算的扩展机制

## 注册新的因子
功能说明：用户可以指定创建一个新的因子（包括因子模型中的名称、配置、计算逻辑等等），若因子是新创建的，则落库存储。否则，忽略。

## 注册新的数据源
功能说明：用户可以创建一个新的数据源，数据源中留下待填参数，可以用于生成实例，一个实例是能够被直接执行，并获取出DF格式的原始数据

## 注册因子计算调度任务
功能说明：用户可以配置因子计算调度任务，在满足调度条件下会自动执行，抓取数据，并做因子计算，最终存储落库
创建步骤说明：
1. 创建调度任务，有多种常见形式，如 定时执行、一次性执行等，待配置参数有数据源实例、调度参数、因子参数转换逻辑
2. 创建数据源实例 
   - 选择数据源，写入必要参数，如：`symbol`、开始/结束时间等
   - 得到数据源实例
3. 填写因子参数转换逻辑，如 将原始数据的中的 df['price'], 名称转为'pce', 类型转为整形。 支持多个参数转换。
3. 填写调度参数
   - 若是定时任务则，则可以按照 cron格式进行定时配置
